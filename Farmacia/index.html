Productos Farmacéuticos

Acetaminofen

El Acetaminofén, solo o combinado, se utiliza habitualmente para bajar la fiebre y aliviar el dolor de cabeza, muscular, menstrual, de garganta, de muelas y de espalda.

Ibuprofeno

El ibuprofeno de venta libre se usa para reducir la fiebre y aliviar dolores menores, cefaleas, dolor muscular, artritis, periodos menstruales, resfriado común, dolor de muelas y dolor de espalda.

Nuestros productos están diseñados para brindar atención médica integral.
